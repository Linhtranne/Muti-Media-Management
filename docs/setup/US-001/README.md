# US-001 Setup Package

This folder contains implementation artifacts for building the US-001 Control Plane:

- `airtable-build-spec.json`: canonical Airtable base/table/field/view/automation spec.
- `airtable-manual-runbook.md`: step-by-step manual build instructions.
- `notion-campaign-brief-template.md`: Notion Campaign Brief template body.
- `manual-acceptance-tests.md`: tests to run after Airtable/Notion setup.

US-001 is external configuration work. The repository can define the canonical setup, but actual completion requires evidence in `docs/plans/US-001/US-001-implementation-completion-gate.md`.
